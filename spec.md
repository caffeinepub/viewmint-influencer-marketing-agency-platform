# Viewmint - Influencer Marketing Agency Platform

## Overview
An interactive website for Viewmint, an influencer marketing agency that connects influencers with brands through a comprehensive platform featuring marketing pages and a connection system.

## Core Features

### Marketing Website
- **Landing Page**: Stylish homepage showcasing Viewmint's brand identity and value proposition
- **About Section**: Information about the agency, mission, and team
- **Services Section**: Overview of services offered to influencers and brands
- **Portfolio Section**: Showcase of successful campaigns and collaborations
- **Contact Section**: Contact form and company information

### Connection Platform
- **User Registration**: Separate registration flows for influencers and brands
- **Profile Creation**: 
  - Influencers can create profiles with social media metrics, content categories, and rates
  - Brands can create profiles with company information, campaign requirements, and budgets
- **Opportunity Browsing**: 
  - Influencers can browse brand collaboration opportunities
  - Brands can browse influencer profiles and portfolios
- **Collaboration Requests**: Users can send and receive collaboration requests with messaging capabilities

### Admin Dashboard
- **User Management**: View and manage influencer and brand profiles
- **Listing Management**: Approve, edit, or remove user listings
- **Platform Analytics**: Overview of platform activity and user engagement

## Data Storage (Backend)
- **User Profiles**: Store influencer and brand profile information, authentication data
- **Collaboration Requests**: Store request details, status, and messaging history
- **Contact Form Submissions**: Store inquiries from the contact section
- **Admin Settings**: Store platform configuration and admin user data

## Technical Requirements
- Responsive design optimized for mobile and desktop devices
- Clean, modern interface reflecting professional agency branding
- Secure user authentication and profile management
- Admin-only access controls for dashboard functionality
